# **App Name**: Vettify MCI Compliance View

## Core Features:

- Owner Compliance Dashboard: Dashboard for individual vehicle owners displaying key compliance metrics.
- Association Compliance View: Compliance overview for taxi associations showing the status of member vehicles.
- Fleet Compliance Management: Fleet management compliance dashboard, focused on highlighting expiring documentation or vehicles with compliance violations
- Secure API Authentication: Secure user authentication using OAuth2 or JWT for API access.
- Manual Data Upload: Allow users to manually upload compliance-related documents via CSV.
- AI Compliance Summarizer: AI tool to summarize compliance violations based on available data to explain violation reasons in easy to understand language.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to evoke trust, stability, and authority within the compliance context.
- Background color: Light Blue Gray (#E8EAF6) to provide a clean, professional, and non-distracting backdrop.
- Accent color: Orange (#FF9800) to draw attention to important actions and notifications within the interface.
- Body and headline font: 'Inter', a sans-serif font offering a modern and neutral appearance suitable for both headlines and body text, ensuring readability and consistency across the platform.
- Use consistent and clear icons from a standard library (e.g., Material Icons) to represent compliance statuses, document types, and actions.
- Employ a clear and structured layout with a consistent grid system to ensure data is easily readable and the user interface is intuitive.
